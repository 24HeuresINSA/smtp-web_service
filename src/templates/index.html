This web service permit to simply send mail with http post request

There is tree entry point :

  • /sendemail : a POST http request take json data like : { "From": "mail@24heures.org", "To": [ "mail@domain.net", "foo@bar.net" ], "Subject": "Testing mail", "Body": "Hello world !" }
  • /verifyemail: a POST http request take list date like : [ "mail@domain.net", "foo@bar.net" ]
  • /getemaillist : a GET http request return list of available sender mails
